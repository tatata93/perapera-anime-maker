# ぺらぺらアニメ作り機

個人制作向けの本格手描き2Dアニメ制作ソフトです。

## 目的

- 手描き2Dアニメを中心に制作する
- 3Dは完成映像ではなく、構図・パース・動きの作画補助として使う
- 常時60fpsを目標に、軽い描画コアを段階的に作る

## 技術スタック

- C++20
- CMake + Ninja
- SDL3
- Dear ImGui
- nlohmann/json

## Phase 0 の状態

この段階では、アプリの土台だけを作っています。

- SDL3ウィンドウを開く
- Dear ImGuiを初期化する
- ダークテーマを適用する
- ①〜⑤のモードタブを切り替える
- 各モードに空のパネルを表示する

作画機能・保存機能・書き出し機能は Phase 1 以降で追加します。

# Inbetween Workflow Policy v1

## 方針

perapera-anime-maker の中割は、最初から自動生成機能として扱わない。
中割は「前後原画の間を埋める作画F」として扱い、手描き中割を速く、分かりやすくするUIを優先する。

## 標準作業手順

```text
1. 原画を描く
2. 原画をタイムシートTに置く
3. 原画間を選ぶ
4. 中割を追加する
5. 前後原画を見ながら中割を描く
6. 原画間だけを再生して確認する
```

## 中割の表示解決

原画・中割・通常作画は、表示解決上はすべて「作画Fを表示する」という同じ動作である。

```text
T2 = 作画F5 / 役割: 中割
```

なら、T2で表示される絵は作画F5である。

## 中割作画補助

中割Fを編集対象にした場合、次の補助が必要。

- 前原画の自動表示
- 次原画の自動表示
- 前後原画の透明度調整
- 中割位置の表示
- 原画間ループ再生
- 中割だけを一覧表示

## 自動中割生成は後回し

自動中割生成は、かなり難しい。
まずは手描き中割の作業を整理し、その後に必要なら補間・変形ガイド・AI支援を検討する。

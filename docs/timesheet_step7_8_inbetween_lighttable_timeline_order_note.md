# Timesheet Rebuild Step 7.8: 中割ライトテーブル候補と再生順タイムライン

## 目的

「原画を先に置く → その間に中割を作る → 前後原画を見ながら中割を描く」を自然に行えるようにする。

また、下部タイムラインの作画F表示を単純な作画F番号順だけでなく、タイムシートに記入された再生順に合わせる。

## 変更内容

- 原画間検出ウィンドウに「中割ライトテーブル」セクションを追加。
- 前原画・次原画の作画F番号を候補として表示。
- 「前後原画をライトテーブルに反映」ボタンを追加。
- ボタンを押すと、前原画・次原画をライトテーブル対象に設定する。
- 「ライトテーブルを消す」ボタンを追加。
- 下部タイムラインの作画Fボックスを、タイムシートに記入されたT順で並べる。
- タイムシートに出てこない作画Fは後ろに追加表示する。

## 方針

中割は自動生成画像ではない。
中割は新しい作画Fとして作成し、前後原画を見ながら人が描く。

そのため、この段階で重要なのは画像補間ではなく、次の作業を簡単にすることである。

1. 原画間を検出する。
2. nコマ打ちで中割Fをタイムシートに置く。
3. 中割Fを編集対象にする。
4. 前後原画をライトテーブル候補としてすぐ表示できるようにする。
5. タイムライン上でも再生される順に作画Fを追えるようにする。

## 今回やらないこと

- 自動中割生成。
- AI補間。
- ベクター補間。
- 画像モーフィング。
- 絵コンテ/背景参照プレートの実装。
- PNG/MP4出力反映。

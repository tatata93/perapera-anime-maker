# Timesheet Rebuild Step 7.2: canvas status, timeline boxes, canvas navigation fix

## 目的

Step 7.1 の確認で出た以下の問題を修正する。

- キャンバス左上のタイムシート状態表示が大きすぎる。
- タイムラインのフレーム表示が四角いコマとして見えにくい。
- タイムシート表示Fと編集Fが違うとき、作画入力停止と一緒にキャンバスのズーム/移動まで止まってしまう。

## 変更内容

- キャンバス左上のタイムシート状態表示を小型バッジに戻した。
- `表示セルN` は残し、`TS ON/OFF`, `T`, `編集F`, `entries` と一緒に1行で表示する。
- 表示Fと編集Fが違う場合でも、ズームと中ボタン移動は許可する。
- 表示Fと編集Fが違う場合に止めるのは、左クリック作画/消しゴム/塗りつぶしだけにした。
- タイムラインのフレーム列を `F1`, `F2`, `F3` の四角ボタンとして表示するようにした。

## まだやらないこと

- タイムシート出力
- PNG/MP4へのタイムシート反映
- タイムシート編集のドラッグ範囲入力
- セリフ/カメラ/素材メモ欄

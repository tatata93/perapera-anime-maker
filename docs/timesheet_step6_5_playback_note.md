# Timesheet Rebuild Step 6.5: タイムシートT再生の試作

## 目的

Step 6でタイムシートT選択をキャンバス表示へ反映したため、次にタイムシートTを時間通りに進める再生確認を追加する。

## 方針

- タイムシート再生は `timesheetPanelState_.selectedTimelineFrame` だけを進める。
- `activeFrameIndex_` は作画F編集対象として維持し、勝手に変更しない。
- 既存の指パラ再生 `isPlayingFrames_` とは同時再生しない。
- 表示Fと編集Fが違う場合は、Step 6の入力停止表示を維持する。
- 保存、出力、Project保存とはまだ接続しない。

## 追加UI

`タイムシート再生` ウィンドウを追加する。

- `|<`: T先頭へ移動
- `<`: 前Tへ移動
- `再生 / 停止`: タイムシートT再生を開始/停止
- `>`: 次Tへ移動
- `>|`: T末尾へ移動
- `速度`: 0.25x / 0.5x / 1x / 2x
- `ピンポン`: 端で折り返す

## 今回やらないこと

- タイムシート再生の音声同期
- タイムシート再生の出力反映
- PNG / MP4 出力反映
- Project保存との自動連動
- セリフ、カメラ、撮影、素材メモ欄の再生表示

## 確認項目

- 再生ボタンでTが進む。
- Tごとの作画指定に従ってキャンバス表示が切り替わる。
- 作画F編集対象は変わらない。
- 表示Fと編集Fが違う場合、キャンバス入力停止警告が出る。
- 指パラ再生とタイムシート再生が同時に走らない。

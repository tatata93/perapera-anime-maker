# Timesheet Rebuild Step 6: タイムシートT選択のキャンバス表示反映

## 目的

Step 6では、タイムシートに入力された作画指定を `TimesheetResolver` で解決し、選択中のTをキャンバス表示へ反映する。

## 方針

- タイムシートに1件以上入力がある場合だけ、タイムシート表示プレビューを有効にする。
- `activeFrameIndex_` は作画F編集対象として維持する。
- T選択によって `activeFrameIndex_` を自動変更しない。
- 表示Fと編集Fが違う場合は、誤編集防止のためキャンバス入力を一時停止する。
- 再生、PNG/MP4出力、Project保存の自動連動はまだ行わない。

## 確認ポイント

- Tごとに指定した作画Fがキャンバス表示に反映される。
- 空セル指定では表示されない。
- ホールド指定では前の表示状態が維持される。
- 作画F編集対象とタイムシート表示対象が違う場合、キャンバスに警告が出て入力が止まる。

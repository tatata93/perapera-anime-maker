# Timesheet Rebuild Step 7.10: タイムシート補助UI整理と作画対象同期

## 目的

タイムシートTで見えている作画Fと、下部タイムラインで選んでいる編集対象の作画Fが一致しない場合に、作画入力が止まってしまう問題を解消する。

## 方針

- Tはタイムシート上の時間位置。
- 作画Fは実際に描く絵。
- タイムシートT選択時は、可能ならそのT/セルで表示される作画Fを編集対象へ自動追従する。
- 下部タイムラインで作画Fを選んだ場合は、その作画Fを編集対象として優先する。
- タイムシート表示と編集対象Fがズレても、作画を止めない。
- ズレている場合は、タイムシート表示の上に編集中作画Fを重ねて表示し、描ける状態を維持する。

## 追加UI

`タイムシート再生` ウィンドウ内に `作画対象` セクションを追加した。

- `T選択で編集対象も追従`
  - タイムシートのT/セルを選んだ時、表示される作画Fへ編集対象を寄せる。
- `今のTの作画Fを編集`
  - 現在選択中T/セルの表示作画Fへ手動で編集対象を合わせる。
- `ズレても編集中Fを上に表示して描く`
  - タイムラインで選んだ作画Fをタイムシート表示上に重ね、作画可能にする。

## 変更点

- 表示Fと編集Fが違うだけではキャンバス入力を止めない。
- mismatch時の警告文を「作画入力停止」から「編集中Fを上に表示して作画可」に変更。
- T/セル選択変更時に、設定がONなら編集対象を自動同期する。

- Project保存との正式連動。
- Scene Plate実装。
- タイムシート補助ウィンドウの完全統合。

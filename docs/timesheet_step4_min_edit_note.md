# Timesheet Rebuild Step 4: 最小編集ノート

このStepでは、正式タイムシートUIに最小編集操作を追加した。

## できること

- タイムシート表のセル列マスをクリックして選択する
- 選択中のTとセル列を表示する
- 作画F番号を入力する
- 選択マスへ以下を入力する
  - 作画
  - 原画
  - 中割
  - ホールド
  - 空セル
  - 消去
- 入力結果をタイムシート表に表示する

## 表示記号

```text
数字      = 通常の作画F指定
原+数字   = 原画としての作画F指定
中+数字   = 中割としての作画F指定
｜        = ホールド
×         = 空セル
—         = 未記入
□         = 選択中の未記入マス
```

## まだやらないこと

このStepの入力結果は `TimesheetPanelState::entries` 内にだけ保持する一時データである。
まだ以下には接続しない。

- Project / Cut / Timesheet 本体
- timesheet.json 保存
- cut.json 保存
- キャンバス表示反映
- 再生反映
- PNG / MP4 出力反映

## 理由

前回の仮タイムシートでは、UI、保存、表示反映、出力を同時に接続したことで不安定になった。
そのため、正式実装ではまずUI上の編集操作だけを独立して確認する。

# Timesheet Rebuild Step 6.6: タイムライン再生表示優先修正

## 目的

Step 6/6.5 でタイムシートT表示をキャンバスへ反映した結果、タイムシート入力がある状態では通常の指パラ/タイムライン再生で `activeFrameIndex_` が進んでも、キャンバス表示がタイムシートT側に固定されて動いていないように見える問題が出た。

## 変更

- タイムシート入力が存在していても、`isPlayingFrames_` が true の間はタイムシートプレビュー表示を一時停止する。
- 通常の指パラ/タイムライン再生中は `activeFrameIndex_` の表示を優先する。
- タイムシート再生は従来どおり、タイムシートTだけを進める。
- 作画F編集対象とタイムシートTは混ぜない方針を維持する。

## 今回やらないこと

- タイムシート再生と通常タイムライン再生の統合
- PNG/MP4出力反映
- Project保存との自動連動
- セリフ/カメラ/撮影/素材メモ欄の再生同期

## 確認

1. タイムシート入力がある状態で、タイムシート再生はTに従って表示が切り替わる。
2. 通常の指パラ/タイムライン再生では、作画Fが進んでキャンバス表示も動く。
3. 通常再生停止後は、再び選択中Tのタイムシート表示へ戻る。

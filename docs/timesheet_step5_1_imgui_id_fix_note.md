# Timesheet Rebuild Step 5.1: ImGui ID衝突修正

## 目的

タイムシート表のマスをドラッグしたときに Dear ImGui の赤い警告ウィンドウが出る問題を修正する。

## 原因

タイムシート表の各マスは、未記入なら `—`、選択中なら `□`、ホールドなら `｜` のように、複数行・複数列で同じ表示ラベルを持つ。

Step 4時点ではセル列だけを `PushID(cellColumn)` していたため、同じ列の別T行にある同じラベルのマスが同一IDとして扱われ、ドラッグ時に Dear ImGui がID衝突を検出していた。

## 修正内容

`TimesheetPanel.cpp` のマス描画で、`PushID(frame)` と `PushID(cellColumn)` の両方を使い、T行とセル列を合わせた一意IDにした。

## 今回やらないこと

- タイムシート保存
- キャンバス表示反映
- 再生・出力反映
- ドラッグ範囲入力
- 複数マス選択

## 確認項目

- タイムシートの未記入マス上をドラッグしても赤い Dear ImGui ID警告が出ない
- マスをクリックして選択できる
- 作画 / 原画 / 中割 / ホールド / 空セル / 消去が従来どおり動く
- フレーム追加・複製・削除で落ちない
- 彩色モード移行で落ちない

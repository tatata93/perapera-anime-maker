# Timesheet Rebuild Step 7.9: T範囲再生と再生順オニオン

## 目的

原画間だけを特別扱いせず、任意のタイムシートT範囲を再生できるようにする。
原画間再生は、T範囲再生のプリセットとして扱う。

また、タイムシート入力がある場合、オニオンフレームは作画F番号順ではなく、タイムシート上の再生順を優先する。

## 用語

- T: タイムシート上の時間位置。
- 作画F: 実際に描く絵の番号。
- コマ数: T上の持続間隔。
- フレーム: 内部処理・出力処理上の単位。UIではなるべく単独で使わない。

## 変更内容

- タイムシート再生ウィンドウに T範囲再生 を追加した。
- 開始T / 終了T を指定できるようにした。
- 範囲再生は開始T〜終了Tの中だけでTを進める。
- ピンポン再生時は範囲端で反転する。
- 全Tを範囲にするボタンを追加した。
- 原画間検出ウィンドウに、この原画間を再生T範囲へ設定するボタンを追加した。
- タイムシート入力がある場合、オニオンの前後候補はT順の再生順から求めるようにした。
- 再生順オニオンの前後作画Fを原画間検出ウィンドウへ表示した。

## 例

タイムシートが以下の場合:

```text
T1 = 作画F1 / 原画
T3 = 作画F3 / 中割
T5 = 作画F4 / 中割
T7 = 作画F2 / 原画
```

作画F番号順は `F1 -> F2 -> F3 -> F4` だが、タイムシート再生順は以下になる。

```text
F1 -> F3 -> F4 -> F2
```

そのため、作画F3を編集中のオニオン候補は:

```text
前: 作画F1
次: 作画F4
```

作画F4を編集中のオニオン候補は:

```text
前: 作画F3
次: 作画F2
```

# Timesheet Rebuild Step 7.7: nコマ打ち中割配置と作画F自動作成

## 目的

原画を先に置き、その原画間へ中割を追加するとき、`中割を何枚追加` ではなく `1コマ打ち / 2コマ打ち / 3コマ打ち / nコマ打ち` で指定できるようにする。

また、タイムシートで存在しない作画F番号を指定した場合、その作画Fを手動で別パネルから追加しなくてもよいよう、対象セルに不足分の空作画Fを自動作成する。

## 用語

- `T`: タイムシート上の時間位置。
- `作画F`: セル内で実際に描く絵の番号。
- `nコマ打ち`: 原画Tから nT ごとに中割用の作画Fを置く指定。
- `原画`: 作画Fの役割。
- `中割`: 前後原画の間を埋める作画Fの役割。

## nコマ打ち配置

例: `T1 = 原画 作画F1`, `T7 = 原画 作画F2` の場合。

- `1コマ打ち`: T2/T3/T4/T5/T6 に中割を置く。
- `2コマ打ち`: T3/T5 に中割を置く。
- `3コマ打ち`: T4 に中割を置く。

既に埋まっているTは壊さず飛ばす。追加できるTが無い場合は警告に留め、原画や既存中割を上書きしない。

## 作画F自動作成

タイムシートの選択マスで `作画 / 原画 / 中割` を指定したとき、指定した作画F番号が対象セルに存在しなければ、対象セルに空の作画Fを不足分だけ追加する。

例: セルAに作画F1しかない状態で、タイムシートに `作画F3 / 原画` を置いた場合、作画F2と作画F3を空で作成する。

自動作成は明示的にタイムシートへ `作画 / 原画 / 中割` を置いた場合だけ行う。`ホールド / 空セル / 消去` では作画Fを作成しない。
